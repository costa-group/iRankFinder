Checking Assertions

We achieve similar refinements for the examples of GulwaniJK09. And we could handle all examples of SharmaDDA11. You can reproduce this on the iRankFinder's web interface. With the profile of settings "Check Invariants".

To check the results we have translate the c source on to an ITS. And we have added the assertions on the ITSs so that we can compare the computed invariant with the assertion. When executing one example the first result will be the list of nodes of the original ITSs where the assertions holds or not and then a second list with the nodes of the refined graph.
