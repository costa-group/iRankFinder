iRankFinder

iRankFinder is open-source and available on github.
Repositories:

  • Information and installation scripts
  • Polyhedra and solvers interfaces
  • Generic parser for C and several ITS formats
  • Main iRankFinder repository
  • Partial Evaluation for Horn Clauses

A Statically linked version is available here .
You can run irankfinder --help to see the details of the parameters of the termination analyser.
Or you can use only the CFR tool with CFRefinement --help.

And an online interface is accessible (via EasyInterface) from here.
We also give direct access to the control flow refinement tool.
